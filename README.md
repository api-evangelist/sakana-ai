# Sakana AI (sakana-ai)

Sakana AI is a Tokyo-based AI R&D company founded by ex-Google researchers focused on collective intelligence, evolutionary optimization, and nature-inspired AI methods. The company's first commercial API product is Sakana Fugu - a multi-agent orchestration system that routes tasks across frontier foundation models (currently in beta).
